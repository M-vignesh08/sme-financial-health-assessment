# SME Financial Health Assessment Platform

This project is a proof-of-concept AI-powered platform designed to assess the financial health of Small and Medium Enterprises (SMEs).

## Key Capabilities
- Financial statement and cash flow analysis
- Creditworthiness and financial risk assessment
- AI-driven insights and recommendations
- Industry benchmarking and financial forecasting
- Tax compliance checks (GST)
- Working capital optimization
- Investor-ready financial reports
- Multilingual support (English + Hindi)

> Note: Banking and GST integrations are simulated using uploaded data files for hackathon evaluation.

## Tech Stack
- Backend: Python (FastAPI)
- Data Processing: pandas
- AI Layer: GPT-based narrative generation
- Frontend: React.js
- Database: PostgreSQL (POC mode)
