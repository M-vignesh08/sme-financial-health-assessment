# **App Name**: FinSight Advisor

## Core Features:

- Financial Statement Analysis: AI-driven analysis of uploaded financial statements (CSV, XLSX, PDF) to extract key financial metrics.
- Cash Flow Pattern Identification: Identify patterns in cash flow data using AI, predicting potential liquidity issues.
- Creditworthiness Evaluation: Evaluate SME creditworthiness based on financial data and business metrics, generating a credit score.
- Personalized Recommendations: Recommend actionable insights, cost optimization strategies, and suitable financial products. LLM uses business metrics to choose which financial planning 'tool' to suggest to the user.
- Automated Bookkeeping Assistance: Provide assistance in bookkeeping by classifying transactions and categorizing expenses.
- Financial Forecasting: Predict future financial performance based on historical data and market trends using AI.
- GST & Banking API Integration: Securely integrate with GST portals and banking APIs to fetch and synchronize financial data.

## Style Guidelines:

- Primary color: Deep blue (#2E4DA7), evoking trust and reliability.
- Background color: Very light blue (#EBF0FF), a desaturated tint of the primary color for a clean backdrop.
- Accent color: Vibrant orange (#E07A39), used sparingly for CTAs and key highlights.
- Headline font: 'Space Grotesk' sans-serif for headlines, body text 'Inter' sans-serif.
- Use a consistent set of financial icons that are clean, modern, and easily recognizable.
- Design a dashboard layout that provides a clear overview of financial metrics with drill-down options.
- Implement subtle animations and transitions to enhance user engagement without being distracting.
